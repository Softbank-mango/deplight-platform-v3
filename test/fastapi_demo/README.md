# FastAPI Demo Application

A simple FastAPI application for testing the Deplight Platform deployment pipeline.

## Features
- Simple REST API
- Health check endpoint
- Fast startup time
- Minimal dependencies

## Endpoints
- `GET /` - Hello world
- `GET /health` - Health check
- `GET /api/status` - API status
- `GET /docs` - Swagger UI

## Local Development
```bash
pip install -r requirements.txt
python main.py
```

## Deployment
This app is designed to be deployed automatically by Deplight Platform with zero configuration!
